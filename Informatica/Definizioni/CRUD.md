- Le quattro operazioni fondamentali di manipolazione dei dati
- Create, Read, Update, Delete

| Metodo HTTP | Equivalente CRUD | Equivalente SQL |
| ----------- | ---------------- | --------------- |
| GET         | READ             | SELECT          |
| POST        | CREATE           | INSERT          |
| PUT         | UPDATE           | UPDATE          |
| DELETE      | DELETE           | DELETE          |
